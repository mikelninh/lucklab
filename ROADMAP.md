## The decision filter

Every new feature must pass:

1. Does it serve the core user (someone stuck, looking for clarity)?
2. Can the user name what they got from it?
3. Does it survive the Hara material-honesty test (no decoration,
   no theatre, no invented social proof)?

If a feature fails any of the three, it waits.
